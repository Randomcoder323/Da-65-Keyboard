# Da-65-Keyboard

## What
This is my first full keyboard, and I chose for it to be 65%. It features 67 keys with hotswap. A diode for each each along with a RGB led on the bottom. The 3d print requires no hardware. I added a level changer for the leds too! Its powered by an esp32 for lots of power. I had so much fun making this and learned lots of new stuff about electronics and pcbs.
## Why
I made this to improve my pcb making skills. I used lots of new items like the stablizers and how to layout siwtches succesfully.
